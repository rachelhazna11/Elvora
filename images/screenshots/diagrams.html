Alur Sistem End-to-End — AI Style Match

«activity diagram» — swimlane per layer
Browser — style-match.js
Supabase Edge Function
Gemini 1.5 Flash (Vision)
Browser — Alpine.js UI
1. User upload foto diri
2. Canvas resize — max-edge: 800px
3. Upload ke Supabase Storage
bucket: user-uploads (private)
path: {uid}/{session}/photo
4. Generate signed URL (TTL: 60s)
5. Verifikasi JWT user
tolak jika 401 Unauthorized
6. Fetch foto via signed URL
konversi ke base64 (inline_data)
7. Kirim ke Gemini API:
— foto (base64 inline_data)
— 22 produk katalog (JSON)
— preferensi user
8. Parse JSON response Gemini
9. Simpan ke ai_style_sessions
(non-fatal — tidak blok response)
— Analisis visual foto:
undertone, proporsi, estetika
— Mapping ke 22 hero products
berdasarkan aesthetic_tags
dan primary_color
— Output JSON:
{ colour_guidance,
 recommendations: [{
  name, why_it_works,
  product_ids[] }] }
10. Terima product_ids[] dari recommendations
11. Query Supabase products
WHERE id IN (product_ids[])
→ fetch nama, gambar, harga
12. Render outfit cards
+ "Why it works" (Elvy)
+ colour_guidance panel
