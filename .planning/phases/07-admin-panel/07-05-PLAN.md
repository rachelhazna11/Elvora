# Plan 07-05: Testimonials + Homepage Content Management

## Goal

Admin dapat mengelola testimonial (CRUD + visibility toggle) dan menandai produk sebagai best sellers — perubahan langsung terlihat di homepage tanpa redeploy.

## Context

Plan 07-01 membuat Testimonials section dan Content section di `admin.html` dengan placeholder text. Plan 07-02 telah memfix best sellers query di `index.html` agar menggunakan filter `is_best_seller=true`. Plan ini mengisi UI lengkap untuk kedua section tersebut.

**Testimonials schema** (dari 001_schema.sql):
- `id uuid, customer_name text, quote text, activity_label text, display_order integer, is_active boolean`
- RLS: `testimonials_admin_all` (INSERT/UPDATE/DELETE/SELECT untuk admin), `testimonials_public_read` (is_active=true saja)
- Admin query HARUS TANPA filter `is_active` agar melihat semua testimonial

**Best sellers** dikelola via toggle pada `products.is_best_seller boolean`. Tidak ada tabel terpisah. Admin melihat semua produk dengan checkbox, update langsung via `supabase.from('products').update({is_best_seller: bool}).eq('id', id)`.

Plan ini bisa berjalan paralel dengan 07-03 dan 07-04 karena menyentuh section berbeda di `admin.html` dan fungsi berbeda di `js/admin.js`.

### Task 3: Implement Testimonials + Content Functions di js/admin.js

**File yang dimodifikasi:** `js/admin.js`

**A. Ganti placeholder `adminGetTestimonials()` dengan implementasi penuh:**
```javascript
async function adminGetTestimonials() {
  // NO is_active filter — admin sees ALL testimonials including inactive
  const { data, error } = await window.supabase
    .from('testimonials')
    .select('*')
    .order('display_order', { ascending: true });
  if (error) throw error;
  return data || [];
}
```

**B. Tambahkan fungsi testimonials baru:**
```javascript
async function adminSaveTestimonial(testimonialData) {
  const payload = {
    customer_name: testimonialData.customer_name.trim(),
    quote: testimonialData.quote.trim(),
    activity_label: testimonialData.activity_label || null,
    display_order: Number(testimonialData.display_order) || 0,
    is_active: testimonialData.is_active !== false,
  };

  if (testimonialData.id) {
    const { error } = await window.supabase
      .from('testimonials')
      .update(payload)
      .eq('id', testimonialData.id);
    if (error) throw error;
  } else {
    const { error } = await window.supabase
      .from('testimonials')
      .insert(payload);
    if (error) throw error;
  }
}

async function adminToggleTestimonial(id, isActive) {
  const { error } = await window.supabase
    .from('testimonials')
    .update({ is_active: isActive })
    .eq('id', id);
  if (error) throw error;
}

async function adminDeleteTestimonial(id) {
  const { error } = await window.supabase
    .from('testimonials')
    .delete()
    .eq('id', id);
  if (error) throw error;
}
```

**C. Ganti placeholder `adminGetCollections()` dan tambahkan `adminSetBestSeller()`:**
```javascript
async function adminGetCollections() {
  const { data, error } = await window.supabase
    .from('collections')
    .select('id, name, slug, is_featured, display_order')
    .order('display_order', { ascending: true });
  if (error) throw error;
  return data || [];
}

async function adminSetBestSeller(productId, isBestSeller) {
  const { error } = await window.supabase
    .from('products')
    .update({ is_best_seller: isBestSeller })
    .eq('id', productId);
  if (error) throw error;
}
```

`saveTestimonial()`:
```javascript
async saveTestimonial() {
  if (!this.testimonialForm.customer_name.trim()) { showToast('Nama author wajib diisi', 'error'); return; }
  if (!this.testimonialForm.quote.trim()) { showToast('Quote wajib diisi', 'error'); return; }
  this.testimonialFormSaving = true;
  try {
    await adminSaveTestimonial(this.testimonialForm);
    showToast(this.editingTestimonial ? 'Testimonial diperbarui' : 'Testimonial ditambahkan');
    this.showTestimonialForm = false;
    this.resetTestimonialForm();
    this.testimonials = await adminGetTestimonials();
  } catch (err) {
    showToast('Gagal menyimpan: ' + err.message, 'error');
  } finally {
    this.testimonialFormSaving = false;
  }
},
```

`toggleTestimonialVisibility(id, isActive)`:
```javascript
async toggleTestimonialVisibility(id, isActive) {
  try {
    await adminToggleTestimonial(id, isActive);
    // Update local state tanpa re-fetch
    const t = this.testimonials.find(t => t.id === id);
    if (t) t.is_active = isActive;
    this.testimonials = [...this.testimonials];
    showToast(isActive ? 'Testimonial ditampilkan' : 'Testimonial disembunyikan');
  } catch (err) {
    showToast('Gagal update: ' + err.message, 'error');
    // Revert checkbox — refresh dari server
    this.testimonials = await adminGetTestimonials();
  }
},
```

`deleteTestimonial(id, name)`:

async deleteTestimonial(id, name) {

## Verification

**Testimonials:**
1. Klik "Testimonials" di sidebar → tabel tampil dengan semua testimonial (termasuk inactive)
2. Toggle switch pada testimonial inactive → is_active berubah → switch visual update tanpa page reload
3. Buka `index.html` di tab baru → hanya testimonial `is_active=true` yang muncul
4. Klik "+ Add Testimonial" → form modal muncul → isi data → klik "Add Testimonial" → testimonial muncul di list
5. Klik "Edit" → form ter-pre-populate → ubah data → "Save Changes" → list di-refresh
6. Klik "Delete" → konfirmasi browser → testimonial hilang dari list

**Best Sellers:**
1. Klik "Content" di sidebar → daftar semua produk dengan toggle switch dan thumbnail
2. Toggle produk → is_best_seller di Supabase berubah → toast notifikasi
3. Buka `index.html` di tab baru → produk yang di-toggle tampil di section Best Sellers (query sudah difix oleh plan 07-02)
