# Plan 07-01: CSS + Admin Shell + Guard

## Goal

Admin melihat shell terproteksi dengan sidebar navigation berbasis hash routing — unauthenticated user dan non-admin diredirect sebelum melihat konten apapun.

## Context

`admin.html` saat ini hanya stub kosong (body berisi `<!-- Phase 7 fills this -->`). `js/admin.js` hanya stub dengan fungsi kosong. Pola yang diikuti: `account.html` menggunakan hash anchor untuk panel switching; `components.js` mengekspos `window.supabase`; `src/input.css` menggunakan `@layer components {}` untuk semua CSS class (lihat pola Phase 6 CSS di baris 90+ file tersebut).

Admin guard HARUS menggunakan `supabase.auth.getUser()` (server-validated JWT) — BUKAN `getSession()` yang hanya membaca localStorage dan bisa dimanipulasi. Check `user.app_metadata?.role === 'admin'`.

`adminReady: false` di awal Alpine state mencegah flash konten admin sebelum guard selesai. Seluruh admin shell dibungkus `x-show="adminReady"`.

### Task 3: Implement js/admin.js — Guard + Dispatcher + Window Exposure

**File yang dimodifikasi:** `js/admin.js`

Ganti seluruh stub dengan implementasi lengkap. File ini harus:

1. **Export `adminApp()` function** yang merupakan Alpine x-data object dengan semua state dan methods.

2. **`init()` method (admin guard):**
   ```
   async init() {
     const { data: { user }, error } = await window.supabase.auth.getUser();
     if (error || !user) { window.location.replace('/index.html'); return; }
     const role = user.app_metadata?.role;
     if (role !== 'admin') { window.location.replace('/index.html'); return; }
     
     // Guard passed — hide loading screen, show shell
     const loadingEl = document.getElementById('admin-loading-screen');
     if (loadingEl) loadingEl.style.display = 'none';
     this.adminReady = true;
     
     // Set initial section from hash
     const hash = window.location.hash.replace('#', '');
     const validSections = ['products', 'orders', 'content', 'testimonials'];
     this.activeSection = validSections.includes(hash) ? hash : 'products';
     await this.loadSection(this.activeSection);
     
     // Listen for hash changes (browser back/forward)
     window.addEventListener('hashchange', async () => {
       const h = window.location.hash.replace('#', '');
       this.activeSection = validSections.includes(h) ? h : 'products';
       await this.loadSection(this.activeSection);
     });
   }
   ```

3. **`setSection(name)` method:** `window.location.hash = name;` — hashchange handler di init() menangani loading.

4. **`loadSection(name)` dispatcher:**
   - `'products'` → `this.productsLoading = true; this.products = await adminGetProducts(); this.productsLoading = false;`
   - `'orders'` → `this.ordersLoading = true; this.orders = await adminGetOrders(); this.ordersLoading = false;`
   - `'content'` → `this.contentLoading = true; this.bestSellers = await adminGetProducts(); this.collections = await adminGetCollections(); this.contentLoading = false;`
   - `'testimonials'` → `this.testimonialsLoading = true; this.testimonials = await adminGetTestimonials(); this.testimonialsLoading = false;`
   - Setiap load dibungkus try/catch yang memanggil `showToast(err.message, 'error')`

5. **`doSignOut()` method:** `await window.supabase.auth.signOut(); window.location.replace('/index.html');`

6. **`openProductForm(product)` method:** `this.editingProduct = product; this.showProductForm = true;`

7. **`openTestimonialForm(testimonial)` method:** `this.editingTestimonial = testimonial; this.showTestimonialForm = true;`

8. **`toggleOrderItems(orderId)` method:**
   - Jika `this.selectedOrderId === orderId`, set ke null (collapse)
   - Jika berbeda: `this.selectedOrderId = orderId; this.loadingItems = true; this.orderItems = await adminGetOrderItems(orderId); this.loadingItems = false;`
