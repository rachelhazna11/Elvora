# BLOCKING CONSTRAINTS — Read Before Anything Else

- [ ] CONSTRAINT: x-show strips display:flex — use `x-if` for any overlay div that needs `display:flex` for centering. x-show toggles display:none/'' but '' reverts to block, not flex.
- [ ] CONSTRAINT: Fixed elements inside backdrop-filter parent — never put `position:fixed` drawer/modal inside a nav that has `backdrop-blur`. Chrome/WebKit makes it the containing block, clipping the fixed child.

**Do not proceed until all boxes are checked.**

## Critical Anti-Patterns

| Pattern | Description | Severity | Prevention Mechanism |
|---------|-------------|----------|---------------------|
| x-show on flex overlay | x-show removes display:flex when hiding; restores to block not flex — breaks centering | blocking | Use x-if for any overlay/modal that needs display:flex |
| Fixed child of backdrop-filter parent | backdrop-filter creates containing block for fixed descendants in Chrome/WebKit | blocking | Keep drawer/backdrop as siblings of nav, not children |
| Wrong property name in x-data | this.success vs this.submitted — silent failure, no error | advisory | Always match x-show binding names exactly to x-data properties |
