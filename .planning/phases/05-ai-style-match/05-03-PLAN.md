  <action>
    Complete `submitStyleMatch(photoFile, preferences)` in `js/style-match.js`:
    1. If `photoFile` exists, resize it.
    2. Upload the resized image to `user-uploads` bucket via `supabase.storage`.
    3. Generate a signed URL (60s) for the uploaded image.
    4. Call the `style-match` Edge Function via `supabase.functions.invoke` with the signed URL and preferences.
    5. Return the AI results.
  </action>

  </verify>
  <done>Photo is uploaded and Edge Function is called with the results.</done>
</task>
