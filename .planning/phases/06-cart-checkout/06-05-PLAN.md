# Plan 06-05: Checkout Page — 4-Step Machine with Supabase Order Creation

## Goal
`checkout.html` renders a 4-step checkout (Shipping → Review → Payment → Confirmation) controlled by an Alpine x-data state machine; a guest user can complete checkout by providing only an email address; authenticated users have their profile pre-filled; on confirmation an order record is written to Supabase `orders` + `order_items` tables with snapshotted `unit_price`, `product_name`, and `variant_label`, the cart is cleared, and an order ID is displayed.

## Context
`checkout.html` currently has `<main><!-- Phase 6 fills this --></main>` as a stub. It already loads `js/cart.js`, `js/supabase.js`, and `js/components.js`. `window.supabase` is exposed globally by `components.js`. `Alpine.store('auth')` and `Alpine.store('cart')` are both available.

Plan 06-01 added all `.checkout-*`, `.step-dot*`, `.step-connector*`, `.checkout-form-group`, `.checkout-form-row`, `.checkout-field-error`, `.checkout-confirm-icon`, `.payment-card-mock`, `.payment-secure-note` CSS classes.

All checkout logic lives in the `x-data` inline object on `<main>` — no separate JS module is needed.

Key RLS rules from RESEARCH.md (strictly enforce these):
- Guest orders: `user_id: null` (explicitly null, NOT undefined, NOT omitted) + `guest_email: this.email`
- Auth orders: `user_id: user.id` + do NOT set `guest_email` (leave undefined/absent)
- Insert sequence: `orders` table first (returns `id`), then `order_items` with that `order_id`
- `auth.js` synthetic email (`username@elvora.local`) must NEVER appear in `guest_email`

The step indicator uses a flat array of step labels: `['Shipping', 'Review', 'Payment', 'Confirmation']`. Steps are 1-indexed (`this.step` is 1, 2, 3, or 4). A dot is `.done` when `i + 1 < step`, `.active` when `i + 1 === step`, `.pending` when `i + 1 > step`. A connector is `.done` when `i + 1 < step`, `.pending` otherwise. Step 4 has no connector after it.

The order summary sidebar is visible at steps 1–3, hidden at step 4 (confirmation). It shows the cart item count + subtotal using `$store.cart` (cart is NOT cleared until `placeOrder()` succeeds and reaches step 4).

```html
<script>
function checkoutData() {
  return {
    step: 1,
    loading: false,
    orderId: null,
    errorMsg: '',

    // Step 1 form fields
    email: '',
    firstName: '',
    lastName: '',
    address: '',
    city: '',
    province: '',
    postalCode: '',
    phone: '',
    errors: {},

    // Step labels for indicator
    stepLabels: ['Shipping', 'Review', 'Payment', 'Confirmation'],

    init() {
      const user = Alpine.store('auth')?.user;
      if (user) {
        this.firstName = user.user_metadata?.first_name || '';
        this.lastName = user.user_metadata?.last_name || '';
        // Do NOT prefill email — synthetic @elvora.local email is not suitable for guest_email
        // Authenticated users do not need email at checkout
      }
    },

    validateStep1() {
      this.errors = {};
      const user = Alpine.store('auth')?.user;
      // Email required only for guests
      if (!user && (!this.email || !/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(this.email))) {
        this.errors.email = 'Valid email required';
      }
      if (!this.firstName.trim()) this.errors.firstName = 'First name required';
      if (!this.lastName.trim()) this.errors.lastName = 'Last name required';
      if (!this.address.trim()) this.errors.address = 'Address required';
      if (!this.city.trim()) this.errors.city = 'City required';
      return Object.keys(this.errors).length === 0;
    },

    nextStep() {
      if (this.step === 1 && !this.validateStep1()) return;
      if (this.step < 3) {
        this.step++;
        window.scrollTo({ top: 0, behavior: 'smooth' });
      }
    },

    prevStep() {
      if (this.step > 1 && this.step < 4) {
        this.step--;
        window.scrollTo({ top: 0, behavior: 'smooth' });
      }
    },

    async placeOrder() {
      this.loading = true;
      this.errorMsg = '';

      try {
        const user = Alpine.store('auth')?.user;
        const cartItems = Alpine.store('cart').items;
        const subtotal = Alpine.store('cart').total;

        if (!cartItems || cartItems.length === 0) {
          this.errorMsg = 'Your bag is empty. Please add items before placing an order.';
          this.loading = false;
          return;
        }

        // Build order payload — guest vs authenticated
        const orderPayload = {
          status: 'confirmed',
          subtotal: subtotal,
          shipping_address: {
            firstName: this.firstName,
            lastName: this.lastName,
            address: this.address,
            city: this.city,
            province: this.province,
            postalCode: this.postalCode,
            phone: this.phone
          }
        };

        if (user) {
          // Authenticated: set user_id, omit guest_email
          orderPayload.user_id = user.id;
        } else {
          // Guest: explicitly null user_id + real email (NEVER synthetic @elvora.local)
          orderPayload.user_id = null;
          orderPayload.guest_email = this.email;
        }

        // Insert order — must come first due to FK constraint on order_items.order_id
        const { data: order, error: orderErr } = await window.supabase
          .from('orders')
          .insert(orderPayload)
          .select('id')
          .single();

        if (orderErr) throw orderErr;

        // Build order_items — snapshot product_name, variant_label, unit_price
        const lineItems = cartItems.map(item => ({
          order_id: order.id,
          product_id: item.productId,
          variant_id: item.variantId || null,
          product_name: item.name,
          variant_label: [item.colour, item.size].filter(Boolean).join(' / ') || null,
          unit_price: item.price,
          quantity: item.qty
        }));

        const { error: itemsErr } = await window.supabase
          .from('order_items')
          .insert(lineItems);

        if (itemsErr) throw itemsErr;

        // Store order ID before clearing cart
        this.orderId = order.id;

        // Clear cart from Alpine store + localStorage
        Alpine.store('cart').clear();

        // If authenticated, also clear Supabase cart_items
        if (user) {
          await window.supabase
            .from('cart_items')
            .delete()
            .eq('user_id', user.id);
        }

        // Advance to confirmation step
        this.step = 4;
        window.scrollTo({ top: 0, behavior: 'smooth' });

      } catch (err) {
        console.error('[checkout] placeOrder error:', err);
        this.errorMsg = 'Something went wrong placing your order. Please try again.';
      } finally {
        this.loading = false;
      }
    }
  };
}
</script>
```
